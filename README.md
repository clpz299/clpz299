# clpz299

Java 后端工程师，具备气象 / 大气环境 / 应急管理 / 水文水利 / 地理信息等领域的软件研发经验。长期聚焦海量数据的挖掘、分析与处理，擅长将业务规则沉淀为稳定、可观测、可扩展的后端服务与数据链路。

**我在做的事**
- 构建面向行业数据的采集、清洗、聚合、指标化与服务化能力（离线/实时）
- 设计高可靠 API 与数据服务接口，支持多源异构数据统一接入与治理
- 关注工程化落地：可观测性、性能优化、故障演练、容量规划、自动化交付

**技术关键词**
- Java 后端：Spring 生态 / 微服务 / 接口与权限 / 任务调度 / 性能与并发
- 数据工程：流式与消息驱动 / 时序与关系型存储 / 数据质量与血缘 / 数据建模
- 平台与运维：Linux / Nginx / CI/CD / 容器与云原生（Kubernetes）

**领域关键词**
- 气象与大气环境：观测/预报/评估相关数据链路与产品化落地
- 应急管理：多源数据融合、态势支撑与业务流程工程化
- 水文水利：水雨情与工程调度相关数据处理与服务化
- 地理信息：空间要素与专题数据服务化对接
